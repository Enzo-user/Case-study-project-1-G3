MACHINE 03

IEEE 754 BINARY64 · DOUBLE-PRECISION FLOATING-POINT MACHINE

GROUP G3_S40
FORMAT 1 · 11 · 52 BITS
ROUNDING GRS / RNE

Decimal → IEEE 754 double-precision

Enter a decimal number (plain or scientific notation) or a special value. The machine shows the stored 64-bit pattern in binary with proper field spacing and in hexadecimal, with the full derivation.

Quick tests:

Rounding method demonstrator

Rounds a decimal or binary number to a target number of digits/bits using all four methods: chopping (truncation, toward zero), round-up (toward +∞), round-down (toward −∞), and round-to-nearest ties-to-even.

Quick tests:

Addition & multiplication with the GRS method

Operands may be decimal or 16-digit IEEE 754 hexadecimal. The machine performs the operation with Guard, Round and Sticky bits (round-to-nearest ties-to-even) and shows every step, then the result in binary, hexadecimal and decimal. Each result is verified bit-for-bit against the host CPU's IEEE 754 hardware.
